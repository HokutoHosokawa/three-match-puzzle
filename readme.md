### Three match puzzle

## マッチスリーパズル
マッチスリーパズルをUnity ECSで実装しました。

### 操作方法  
動かしたいピースの上で左クリックをし、移動させたい方向のピースにカーソルを動かし、移動後にクリックを離してください。  
そうすると、元のピースと隣接する(上下左右それぞれ1マス)ピースが交換されます。ピースは3つ以上揃うと消えます。ピースは消える方向以外には移動できません。
